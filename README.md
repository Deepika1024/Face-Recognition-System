# Face-Recognition-System for attendance management

*A small Python app that uses the webcam and face-recognition to mark attendance in attendance.csv.*

**Requirements**
 - Python 3.8+
 - A webcam
 - See requirements.txt for pip packages

**Quick setup (PowerShell)**
'''
cd "E:\Project_folder"
py -3 -m venv .\venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
'''

**Generating Encodings**
 - Run your encode_faces.py to create encodings.pickle.

**Run**
'python attendance.py'
Press ==q== to quit the webcam window.
